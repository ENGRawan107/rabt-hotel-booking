# Rabt | Hotel Booking

A simple full-stack hotel booking web application built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express

## ✨ Features

- Create a booking
- View all bookings
- Delete a booking
- Prevent duplicate bookings (same room & date)

## 🚀 How it works

Users can enter their name, room number, and date to create a booking.  
The system checks if the room is already booked for the same date before saving.
